# Where is the note?

The note is not a loose file in the root folder. It lives in the `paper/` folder.

Open:

```text
paper/qkernel_note.tex
```

A compiled PDF is also generated as:

```text
paper/qkernel_note.pdf
```

The paper ZIP contains the paper folder contents directly. The development ZIP contains the full repository, including the same files under `paper/`.

If you only see three files in the paper ZIP, use the latest `qkernel_arxiv_note_draft_v0_19.zip` or later; v0.21 explicitly includes the compiled PDF and note-location file.
